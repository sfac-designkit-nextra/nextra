# Layout & Gird

Layout & Gird 컴포넌트는 기본 레이아웃을 구성하는 컴포넌트 입니다. <br />
`containter : 1024px`, `gutter : 20px`, `column : 12(60px)` 로 제작되었으며,<br />
className으로 `col-span-{1-12}`로 column를 구성하실 수 있습니다.

### Default
 

```tsx copy

  <Layout>
    /** 구성할 내용들 */
  </Layout>

```
