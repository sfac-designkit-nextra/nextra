# Introduction
 
스나이퍼팩토리 디자인팀입니다. 디자인시스템에 대해서 잘 아시나요?
디자인시스템이란 디자인과 관련된 모든 것을 통합하는 시스템입니다. 스나이퍼팩토리의 디자인시스템은 개발자와 디자이너가 함께 만들어가는 시스템이며 앱/웹 개발에 필요한 디자인된 모든 위젯과 컴포넌트를 제공하는데 목표를 두고 있습니다.
그럼, 스나이퍼팩토리 디자인시스템에 대해서 알아볼까요?
