# Button
---
Button은 사용자가 클릭 또는 탭하여 특정 동작을 수행할 수 있는 인터페이스 요소입니다. <br />
일반적으로 텍스트로 표시되며 원하는 동작을 수행합니다. <br />
각 버튼에는 `primary`, `destructive`, `secondary`, `outline`, `link`, `ghost` 등 여섯 가지 Variant와<br/>
두가지의 `disabled`, `asChild` 속성이 있습니다.
<br />
<br />

> ❗️ Button 컴포넌트 스타일이 적용되지 않는 경우 

`tailwind.config.js` 파일에서 `preflight` 속성을 `false`로 변경해주세요.

```js filename="tailwind.config.js" /variant/ /text/ copy

module.exports = {
corePlugins: {
    preflight: false,
  },
  ...
}

```

### Using example


버튼안에 `size`와 `weight` 값을 변경 가능합니다.<br />
5종류의 크기와 3종류의 굵기 정도를 조절 가능합니다. <br/>
`Text`컴포넌트와 동일한 크기과 굵기입니다.

```tsx {7-8} copy

  <Button 
    variant="secondary" 
    text="로그인" 
    asChild
    className=""
    textSize="size값"
    textWeight="weight값"
  />

```
