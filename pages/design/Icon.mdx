# Icon
---
Icon 컴포넌트는 name prop에 따라 해당하는 아이콘을 동적으로 불러와 렌더링하는 시스템입니다.
<br />
<br />
### Default
<br />
import Image from 'next/image'

<Image src="/Iconlist.png" alt="Primary" width={500} height={200}  />
```tsx /name/ copy

  <Icon name="정의된 아이콘 이름" />

```
<br />
- width height 조정을 원할 때

```tsx copy

  <Icon 
    name="정의된 아이콘 이름" 
    height={50} // 원하는 사이즈
    width={50} // 원하는 사이즈
  />

```

import style from "./icon.module.css";

export const IconList = () => {
  return (
    <>
      {Icons.map(item => {
        const iconName = item.icon.replace('.svg', ''); 
        return (
          <ul key={item.icon} className={style.iconList} >
            <li className={style.iconListLi}>
              <Image
                src={`/${item.icon}`}
                alt={`${iconName} Icon`}
                width={21}
                height={21}
              /> 
            </li>
            <li>{iconName}</li>
          </ul>
        );
      })}
    </>
  );
};

### Icon 저장소

Icon은 Image와 Stroke 별로 폴더 정리 했습니다. <br />
이미지 추가 하실 때 참고 하시길 바랍니다.

import { FileTree } from 'nextra/components'
 
<FileTree>
  <FileTree.Folder name="icons" defaultOpen>
    <FileTree.Folder name="imgIcon">
      <FileTree.File name="icon.svg" />
    </FileTree.Folder>
    <FileTree.Folder name="stroke">
       <FileTree.File name="icon.svg" />
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>
