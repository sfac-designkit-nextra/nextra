# Select
`Select`는 하나의 옵션 UI 요소입니다. <br />
`Main`, `Sub` 모드가 있으며, `active`로 활성화 된 상태로 나타낼 수 있습니다. <br />
`Main`모드시에는 `icon`을 사용, Sub 모드시에는 `checkbox`와 아무것도 없는 두가지의 상태로 나타납니다.
