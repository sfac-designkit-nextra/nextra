# Badge

Badge 컴포넌트는 상황에 따라 다른 badge를 사용할 수 있도록 제공하는 컴포넌트입니다

<div
  style={{
    border: "0.5px solid gray",
    borderRadius: "8px",
    display: "flex",
    justifyContent: "center",
    alignItems: "center",
    padding: "32px",
  }}
>
  <Image src="/Badge.png" alt="Badge" width={500} height={500} />
</div>

```tsx
<Badge>무료강의</Badge>
<Badge color="Secondary">무료강의</Badge>
<Badge color="Outline">무료강의</Badge>
<Badge color="Destructive">무료강의</Badge>
```

Badge 컴포넌트는 span 태그로 되어있으며, <br />
color 속성값으로 원하는 디자인을 선택해 사용할 수 있습니다

기본값은 Primary로 되어있고, 순서대로 Primary, Secondary, Outline, Destructive 입니다
