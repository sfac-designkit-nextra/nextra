# Badge

Badge 컴포넌트는 UI(사용자 인터페이스) 디자인에서 자주 사용되는
작은 정보 표시 요소입니다. 주로 알림, 카운트, 라벨, 상태 등을 사용자에게
알리기 위해 사용됩니다. Badge 컴포넌트는 이메일 앱의 읽지 않은 메일 수,
소셜 미디어 앱의 알림 수, 쇼핑 카트의 아이템 수 등과 같은 정보를
표시하는 데 유용합니다.
