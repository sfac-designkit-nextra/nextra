# Accordion

Accordion 위젯은 일반적으로 제목과 해당 섹션을 토글하는 기능으로 구성됩니다. 사용자가 제목을 클릭하면 해당 섹션이 열리거나 닫히는 동작이 발생합니다. 이를 통해 사용자는 필요한 정보에 쉽게 액세스할 수 있고, 필요하지 않은 정보를 숨길 수 있습니다.
