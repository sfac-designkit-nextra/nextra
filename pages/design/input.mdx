# Input

`Input` 컴포넌트는 사용자로부터 데이터를 입력받기 위한 요소입니다.<br /> 웹 페이지나 애플리케이션에서 텍스트, 비밀번호, 이메일 주소 등의 정보를 입력받기 위해 주로 사용됩니다.
