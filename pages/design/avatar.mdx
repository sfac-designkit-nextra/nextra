# Avatar
`Avatar` 컴포넌트는 사용자를 나타내는 원입니다. <br />일반적으로 사용자의 프로필 이미지와 함께 사용하거나 이미지가 없는 경우 기본 이미지가 출력됩니다.

### Usage example
<br />
<div style={{ border: "0.5px solid gray", borderRadius: "8px", display: "flex", justifyContent: "center", alignItems: "center", padding: "32px" }}>
  <Image src={StyledAvatar} alt="Avatar Component usage example" height={100} />
</div>

```tsx {3-8} copy

  <Avatar
    src="https://picsum.photos/300/300" /** 이미지 경로 */
    size="64" /** 원의 크기 */
    border="true" /** border 유무  */
    borderColor="border-color-blue-600" /** 원하는 색상명 */
    alt="user profile"/** 이미지 설명 */
    className="" /** 추가 스타일링이 필요한 경우 */
  />

```
각 속성들을 위와 같이 커스텀하여 사용,<br />
그 외 스타일링이 필요한 부분은 tailwind css 문법에 맞게 `className`의 속성 값으로 적용
