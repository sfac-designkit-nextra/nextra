# Title
Title 컴포넌트는 디자인 시스템 내에 적용된 제목 폰트 설정들을 사전 적용하여 손쉽게 제목 태그들을 표시할 수 있도록 도와주는 컴포넌트 입니다. 

### Default
 
<Image src={TitleExample} alt="Title Component usage example" height={200} />


```tsx
<Title size={"4xl"}>H1_32_B</Title>
<Title size={"3xl"}>H2_28_B</Title>
<Title size={"2xl"}>H3_24_B</Title>
<Title size={"xl"}>H4_20_B</Title>
```

`size` 속성의 값을 통해 4종류의 제목 폰트를 사용할 수 있습니다. 
그 외 스타일링이 필요한 부분은 `className`에 tailwind css 문법에 맞게 적용하면 됩니다.
