# Textarea

`Textarea` 컴포넌트는 여러 줄의 텍스트를 입력받기 위한 요소입니다. <br />
기본적인 input 요소는 일반적으로 한 줄의 텍스트만 입력받기에 적합하지만, <br />
textarea는 여러 줄의 텍스트 입력을 가능하게 합니다.
