# Toast

토스트(Toast)는 사용자에게 간단한 메시지나 알림을 표시하는 UI 요소입니다. 일반적으로 화면의 가장 위쪽 또는 아래쪽에 작은 팝업 형태로 나타납니다.
또한, 일시적으로 정보를 제공하거나 사용자의 액션을 알리는데 사용됩니다.
각 Toast에는 6가지의 `type`이 있으며 `type`에 따라 사용할 수 있는 `title, text,
button`속성이 있습니다.

<br />

  style={{

    padding: "32px",
  }}
>
  <Image src="/SimpleToast.svg" alt="SimpleToast" width={360} height={60} />

```js

```
